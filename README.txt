the aim of this report is to create a proposal of dashboard of the economy health. This study shows data visualisation by the exploring of data from the economic indicators for countries around the world. The report will be comparing the data of countries in different income group in different regions, but this model can be used for any financial indicators and countries. Data visualisation techniques and process of performing statistical analysis will be discussed. Also, different software such as R, Excel, power BI and tableau are used for cleaning, analysing data and designing dashboard. The results of output will be demonstrated in a single dashboard page that simply can be interpret the data. The dashboard will provide the future trade for economic profile based on historical data.
